## Hexwerks Studios

Hexwerks Studios is a creator-led tabletop studio building dark miniature games, compact tactical systems, and myth-soaked physical worlds.

The studio tone is dark, mythic, grim, handcrafted, physical, tactical, and clear. Hexwerks Studios focuses on painted tables, brutal choices, named fighters, oaths, iron, ash, relics, saints, old gods, and compact rules that put pressure on the player.

## Black Oaths

Black Oaths is a grim Viking-age skirmish game about compact warbands, doom, pagan rites, Christian pressure, relics, saints, blood-debt, and old gods.

Players build compact warbands, command named fighters, pursue oaths, suffer wounds and injuries, fight over relics, faith, and old powers, and carry consequences into future games.

Current status: public playtest. Rules are in development. There is no posted release date, price, final rulebook link, or download link.

## EINVIGI

EINVIGI is a brutal Norse duel game about oath, timing, commitment, distance, and reading the opponent.

Each player commands one warrior inside the killing ground. The duel uses hidden intent, Hazel Staves, action commitment, Wyrd, Momentum, Overreach, wounds, Ring-Out pressure, and the edge of the sacred ground.

Current status: prototype. The rulebook and print components are in development. There is no posted release date, price, final rulebook link, or download link.

## Oaths & Banners

Oaths & Banners is a Dark Age warband game about shieldwalls, banners, rough steel, formation, courage, leadership, and hard battlefield choices.

The game is in development. No public rules download is posted on the website.

## Omens & Ravens

Omens & Ravens is a compact hex-based tactical game concept about portable battles, omens, pressure, and sharp table decisions.

The game is in concepting. No public rules download is posted on the website.

## Hoist the Black

Hoist the Black is a brutal age-of-sail ship combat concept about wind, cannon smoke, boarding, infamy, and command decisions.

The game is in concepting. No public rules download is posted on the website.
